Donate via M-Pesa

Support the Ministry Securely via M-Pesa

Choose one of the two easy options below:

Option 1: Send Money (to Phone Number)

0711879706
  1. Open M-Pesa menu (Safaricom SIM → M-Pesa).
  2. Select Send Money.
  3. Enter phone number: 0711879706
  4. Enter amount (e.g., 1000 KES).
  5. Enter your PIN to confirm.
  6. Receive confirmation SMS.

Option 2: Pay to Till Number (Buy Goods)

5057501
  1. Open M-Pesa menu → Lipa na M-Pesa.
  2. Select Buy Goods and Services.
  3. Enter Till Number: 5057501
  4. Enter amount you wish to donate.
  5. Enter your PIN to confirm.
  6. Receive confirmation SMS.

When using Till Number 5057501, if asked for Account/Reference/Message, write your donation type (e.g., "Tithe - John" or "Building Fund"). For Send Money, you can add a note or inform us separately via WhatsApp.

All transactions are secure and directly received via M-Pesa. Thank you!
